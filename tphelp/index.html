installing third-party plugins, a simple-ish(?) guide
official docs
sunnie's plugin guide
install nodejs
install pnpm by running "npm i -g pnpm" after installing nodejs
install git
installed all that? good.
now run
git clone https://github.com/Vendicated/Vencord
you do this in cmd or powershell on windows, terminal if you're on linux/macOS.
this will make a clone of vencord's source code. git clones will be a folder on the directory your cmd/terminal is open in, ideally open cmd/terminal in some place like documents.
open your cloned "vencord" folder and find the src folder. inside it, make a folder called userplugins.
put your third party plugins inside src/userplugins.
most third party plugins will typically be folders. however, some are .ts/.tsx files that you'll simply just put inside userplugins. do not make folders for .ts/.tsx files.
done? now open a cmd/terminal with the "vencord" folder as cd, then run
pnpm i
and then
pnpm build
and finally (skip this if you use vesktop, see below)
pnpm inject
you're all set!
to use third party plugins on vesktop, simply find "vesktop settings" in your discord settings. scroll down and click "change" on the "vencord location" option, and then select the "dist" folder inside your vencord clone.
*don't ask for support with third party plugins in the server.
*manually compiling vencord will make you unable to update normally.
